#Black–Scholes Option Pricer
A Black–Scholes model for pricing European call options and computing the Greeks with a 3D
visualisation of how option price responds to stock price and volatility (stress testing conditions)

#What it does
Prices European call options using the Black–Scholes formula
Computes the five main Greeks
Illustrates call price, stock price and volatility in a 3D surface 

#Run it
pip install numpy matplotlib scipy
python BS.py
